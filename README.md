# 补丁使用教程
下载相应的补丁放入对应设备的游戏文件夹内，可达到无需代理进行游戏的效果

ps1：PC端如无启动器启动，须先挂代理使用DMM客户端启动，游戏启动成功可关闭代理

ps2：手机端如无登录账号，须先挂代理登录账号，登录完成即可关闭代理，之后启动游戏也可无代理进行

具体详见：https://www.bilibili.com/video/BV1rg411k7CZ/
